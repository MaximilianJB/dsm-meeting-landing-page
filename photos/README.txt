PHOTO INSTRUCTIONS
==================

Place profile photos for all panelists and the concert performer in this folder.

Naming convention: firstname-lastname.jpg (lowercase, hyphenated)
  Example: brian-billion.jpg

Requirements:
  - Format: JPG or PNG
  - Minimum size: 160×160 pixels (displayed at 80px; 2× for retina)
  - Crop: Square (photos are displayed as circles via CSS)

Once a photo is added, update the corresponding src attribute in index.html.
Replace src="photos/placeholder.jpg" with src="photos/firstname-lastname.jpg"

Current photo references in index.html:
  - photos/placeholder.jpg  → used for all panels (replace each one individually)
  - photos/brian-billion.jpg → concert performer (Brian Billion)
